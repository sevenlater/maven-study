日期组件

长日期组件

日期组件，input[type='text']标签添加属性 xz-datepicker，ng-model属性是必须的，默认的日期格式为yyyy-MM-dd HH:mm:ss"
自定义日期格式，直接给xz-datepicker赋值即可，如: xz-datepicker='HH:mm:ss MM-dd-yyyy'

短日期组件(不包含时间)

日期组件，input[type='text']标签添加属性 xz-short-datepicker，ng-model属性是必须的，默认的日期格式为yyyy-MM-dd"
自定义日期格式，直接给xz-short-datepicker赋值即可，如: xz-datepicker='MM-dd-yyyy'
